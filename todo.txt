# Story / LLM stack (historical notes)

Status (2026-04): LangChain path is implemented in the Python server. See server code:

- Prompt / message bridge: `server/src/infrastructure/langchain_messages.py` (ChatPromptTemplate + Runnable + BaseMessage list)
- Provider + invoke/stream: `server/src/infrastructure/langchain_chat.py` (Ollama via `langchain_ollama`, DeepSeek via `langchain_openai` OpenAI-compatible)

# 后续功能拓展（路线图草案，未排期）

## 创作与叙事体验
- 可选「自由对话 / 旁白修正」通道：在保留按钮驱动剧情的前提下，与现有 StoryActions 流统一设计。
- 分支与多结局：章节/节点树、存档点、回溯到某次生成前状态。
- 导出与分享：Markdown / 纯文本 / PDF、带元数据的项目包。
- 版本与对比：同段剧情多次「改写」并排对比、一键回滚到某版。
- 模板与预设：题材模板（悬疑、日常、奇幻等）、一键套用角色/大纲结构。

## AI 与后端能力
- 更多模型与接入：除 Ollama、DeepSeek 外，其它 OpenAI 兼容端点、Anthropic、Azure 等（对齐 `langchain_chat` 工厂）。
- 生成参数 UI：temperature、max tokens、停止词等，支持按会话保存。
- RAG / 设定资料库：用户上传世界观文档，检索后注入 context（存储与切块策略）。
- 多模态（远期）：配图或参考图（需产品与合规评估）。

## 数据、隐私与协作
- 备份与恢复向导：一键导出/导入 SQLite 或加密包。
- 可选云同步：端到端加密、多设备（工作量大，远期）。
- 多 profile：本机多用户配置（不同 API Key、不同故事库路径）。

## 桌面与系统集成
- 系统托盘与后台：最小化到托盘、从托盘恢复。
- 全局快捷键：快速打开窗口或触发「继续生成」（Tauri 能力对齐）。
- 更新与发布：内置检查更新、变更日志（若尚未产品化）。

## 国际化与可访问性
- 更多语言：在现有 en/zh 与 `scripts/check-i18n-keys.mjs` 流程上扩展。
- 无障碍：键盘导航、焦点管理、与 axe / Playwright 规则持续对齐。

## 可观测性与运维
- 结构化日志与诊断包：用户一键导出脱敏日志（Flask 端口/连接类问题）。
- 可选匿名遥测：仅崩溃与使用统计，默认关闭并附合规说明。

## 测试与质量
- E2E 覆盖面：关键故事流、断网/重连、模型切换（`playwright*.config`）。
- 性能与长会话：摘要策略调优、消息列表虚拟化与加载状态。
